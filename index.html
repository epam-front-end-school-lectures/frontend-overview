Plain crossbrowser XHR

function getXmlHttpRequest() {
    var xhr;
        if (window.XMLHttpRequest) {
            xhr = new XMLHttpRequest();
        } else {
            try {
                xhr = new ActiveXObject("Msxml2.XMLHTTP");
            } catch (e) {
            try {
                xhr = new ActiveXObject("Microsoft.XMLHTTP");
            } catch (e) {
                xhr = false;
            }
        }
    }
    return xhr;
}

        return {
            people: [
                // ...
            ]
        }
    },
    render: function() {
        return <ul className="people">

                    <h4>{person.lastName}</h4>

    }
});

Homework

Business description

Create registration page. When user enters page he will see short description of what he will get after registration, form page with those fields: name, email, date of birth, postal index, password and password confirmation, list of countries, checkbox with text "I agree with terms and conditions" and submit button. All fields are required. Email, date of birth, postal index should contain valid data.

Homework requirements

  • Basic level: Create page styling using bootstrap
  • Advanced level: add form validation using AngularJS, summary of all fields is displayed under the submit button and is being updated while user types something. If some field is invalid, error is displayed under it and submit button is disabled.
